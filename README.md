# CodeBlock

Визуальный конструктор программ на основе блоков.

## Запуск

Запустите через **Live Server** в VS Code.

## Функционал

- **Создание программы** — перетаскивание блоков из левой панели на холст.
- **Типы блоков**:
  - Объявление переменных и массивов
  - Присваивание
  - Арифметические операции
  - Логические операции
  - Управляющие конструкции: If-Else, While, For
- **Редактирование**:
  - Изменение значений через поля ввода и выпадающие списки
  - Вложенность блоков (перетаскивание выражений в операнды)
  - Перемещение блоков между разными частями программы
- **Валидация**:
  - Проверка объявления переменных в области видимости
  - Визуальная индикация ошибок (красная подсветка и всплывающие подсказки)
- **Сохранение и загрузка**:
  - Экспорт программы в JSON
  - Импорт из JSON
